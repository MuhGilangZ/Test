Membuat Paragraph

Membuat paragraph dapat dilakukan dengan tag p.

Untuk membuat baris baru gunakan tag br dan untuk membuat garis di baris baru gunakan tag hr.


Selain membuat paragraph di halaman ini akan ditampilkan beberapa tag untuk style font.

Tetapi tidak disarankan untuk formating style dengan html!


1. Bold

Bold berfungsi untuk mentebalkan font. Untuk membuat Bold gunakan tag b


2. Italic

Italic befungsi untuk memiringkan font. Untuk membuat Italic gunakan tag i


3. Underline

Underline berfungsi untuk menggaris bawahi font. Untuk membuat Underline gunakan tag u


4. Strong dan Em

Strong dan em berfungsi untuk menandai kata. Strong akan berupa seperti Bold dan em akan berupa seperti Italic.

Untuk membuat Strong gunakan tag strong dan untuk membuat em gunakan tag em.
